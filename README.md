# 🌌 Neural Universe v4.0 | AI Systems Portfolio

An immersive, high-performance professional artifact designed for the intersection of human cognition and machine intelligence. This is not just a portfolio; it is a technical demonstration of full-stack engineering, AI documentation, and immersive UI design.

## 🚀 Vision
The "Neural Universe" is a tactile, interactive digital identity. It features a neural-stream background, synthesized haptic feedback, and a high-fidelity glassmorphic interface that mirrors the complexity of modern AI systems.

## 🛠️ Tech Stack
- **Backend**: FastAPI (Python) with Jinja2 Templating
- **Frontend**: Vanilla JS, Tailwind CSS, Canvas API (Neural Particle Physics)
- **Audio Engine**: Web Audio API (Synthesized UI Feedback)
- **Deployment**: Vercel-ready Serverless Architecture
- **Infrastructure**: Formspree (Stateless Contact), SQLite (Local Telemetry)

## ✨ Elite Features
- **Neural Sync Sequence**: Cinematic initial loading sequence for data-link authentication.
- **Magnetic 3D UI**: Perspective-mapped cards and buttons that respond to cursor proximity.
- **Neural Terminal**: A functional live emulator for status checks and system interaction.
- **Deep-Tech Modals**: AI-generated architecture diagrams and technical deep-dives for every project.
- **Real-Time Telemetry**: Persistent session tracking and system health monitoring in the footer.

## 📦 Installation & Local Development
1. **Clone the Repository**:
   ```bash
   git clone <your-repo-url>
   cd mukul-portfolio
   ```

2. **Environment Setup**:
   ```bash
   pip install -r requirements.txt
   ```

3. **Asset Configuration**:
   Ensure all generated images are located in the `/assets` directory.

4. **Run Local Server**:
   ```bash
   python main.py
   ```
   Access the universe at `http://localhost:8000`.

## 🌐 Deployment
This repository is optimized for **Vercel**. 
1. Connect your GitHub repository to Vercel.
2. Set your `FORMSPREE_URL` in the environment variables.
3. Deploy. The `vercel.json` will automatically configure the Python serverless functions.
